## Outreach Strategy

### Step 1: Find Contacts

For each prospect, reverse-engineer the decision-makers:

| Role | Where to Find |
|------|--------------|
| **PAC Director / Executive Director** | FEC filings (fec.gov), organization website, LinkedIn |
| **Media Buyer / Ad Buyer** | LinkedIn search "[Org Name] media buyer", look for firms like GMMB, Targeted Victory, etc. |
| **Campaign Manager** | FEC filings, press releases, LinkedIn |
| **Communications Director** | Organization website, LinkedIn |
| **Treasurer** | FEC filings (listed on every filing) |
| **Digital Director** | LinkedIn, organization website |

**Pro tip:** FEC filings at https://www.fec.gov/data/ list the **treasurer** and **contact info** for every registered committee. That's public record.

### Step 2: Build the Spreadsheet

Columns for the prospecting spreadsheet:

| Column | Description |
|--------|-------------|
| Organization | PAC / Committee name |
| Lean | R / D / Bipartisan |

| 2022 Spend | Historical spending (shows consistency) |
| Tier | 1 (Whale) / 2 (Major) / 3 (Mid) |
| Contact Name | Decision-maker name |
| Title | Their role |
| Email | Contact email |
| LinkedIn | Profile URL |
| Status | Not contacted / Emailed / Responded / Meeting / Closed |
| Link Sent | Which landing page URL was sent |
| Notes | Any intel |

## Key Insights

- **Top 5 Super PACs spent $1.4 BILLION** in 2024 alone
- **Video is the #1 line item** for most of these orgs (TV ads + digital video)
- **Both sides spend roughly equally** â€” don't limit to one party
- **Cycle timing matters** â€” reach out 12-18 months before election day for max budget availability
- **Recurring clients:** Organizations like Senate Leadership Fund, Congressional Leadership Fund, and Senate Majority PAC spend $200M+ *every cycle* â€” these are annuity accounts
- **New money:** Crypto PACs (Fairshake, Protect Progress) emerged in 2024 with huge budgets and are likely underserved by established political media firms
- **FEC filings are public** â€” treasurer name, committee address, and contact info are all searchable at fec.gov
